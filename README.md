# 💰 Gestor Financeiro Pessoal (CLI)

## 🎯 Problema Real
Muitos jovens e estudantes têm dificuldade em controlar pequenos gastos diários, o que gera insegurança financeira ao fim do mês.

## 🚀 Solução
Uma aplicação simples e eficiente para registrar receitas e despesas, mantendo o saldo atualizado e persistido em um arquivo JSON para consultas futuras.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** C# (.NET 8)
- **Dados:** JSON
- **Testes:** xUnit
- **CI/CD:** GitHub Actions

## 📦 Como Executar
1. Clone o repositório.
2. No terminal, execute: `dotnet run --project Gerenciador.App`

## 🧪 Como rodar os testes
Execute: `dotnet test`
